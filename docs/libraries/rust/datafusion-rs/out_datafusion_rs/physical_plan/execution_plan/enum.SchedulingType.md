``` rust
pub enum SchedulingType {
    NonCooperative,
    Cooperative,
}
```

Expand description

Represents whether an operator’s `Stream` has been implemented to actively cooperate with the Tokio scheduler or not. Please refer to the [`coop`](https://docs.rs/datafusion/50.2.0/datafusion/physical_plan/coop/index.html "mod datafusion::physical_plan::coop") module for more details.

### NonCooperative

The stream generated by [`execute`](https://docs.rs/datafusion/50.2.0/datafusion/physical_plan/trait.ExecutionPlan.html#tymethod.execute "method datafusion::physical_plan::ExecutionPlan::execute") does not actively participate in cooperative scheduling. This means the implementation of the `Stream` returned by [`ExecutionPlan::execute`](https://docs.rs/datafusion/50.2.0/datafusion/physical_plan/trait.ExecutionPlan.html#tymethod.execute "method datafusion::physical_plan::ExecutionPlan::execute") does not contain explicit task budget consumption such as [`tokio::task::coop::consume_budget`](https://docs.rs/tokio/1.47.1/x86_64-unknown-linux-gnu/tokio/task/coop/consume_budget/fn.consume_budget.html "fn tokio::task::coop::consume_budget::consume_budget").

`NonCooperative` is the default value and is acceptable for most operators. Please refer to the [`coop`](https://docs.rs/datafusion/50.2.0/datafusion/physical_plan/coop/index.html "mod datafusion::physical_plan::coop") module for details on when it may be useful to use `Cooperative` instead.

<a href="https://docs.rs/datafusion/50.2.0/datafusion/physical_plan/execution_plan/enum.SchedulingType.html#variant.Cooperative" class="anchor">§</a>

### Cooperative

The stream generated by [`execute`](https://docs.rs/datafusion/50.2.0/datafusion/physical_plan/trait.ExecutionPlan.html#tymethod.execute "method datafusion::physical_plan::ExecutionPlan::execute") actively participates in cooperative scheduling by consuming task budget when it was able to produce a [`RecordBatch`](https://docs.rs/datafusion/50.2.0/datafusion/common/arrow/array/struct.RecordBatch.html "struct datafusion::common::arrow::array::RecordBatch").

<a href="https://docs.rs/datafusion/50.2.0/datafusion/physical_expr/equivalence/struct.EquivalenceClass.html" class="struct" title="struct datafusion::physical_expr::equivalence::EquivalenceClass">EquivalenceClass</a>  
An `EquivalenceClass` is a set of [`Arc<dyn PhysicalExpr>`](https://doc.rust-lang.org/nightly/alloc/sync/struct.Arc.html "struct alloc::sync::Arc")s that are known to have the same value for all tuples in a relation. These are generated by equality predicates (e.g. `a = b`), typically equi-join conditions and equality conditions in filters.

A collection of distinct `EquivalenceClass`es. This object supports fast lookups of expressions and their equivalence classes.

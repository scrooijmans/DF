## About

The `<Calendar />` component is built on top of the <a href="https://www.bits-ui.com/docs/components/calendar" class="font-medium underline underline-offset-4" rel="noopener noreferrer" target="_blank">Bits UI Calendar</a> component, which uses the <a href="https://react-spectrum.adobe.com/internationalized/date/index.html" class="font-medium underline underline-offset-4" rel="noopener noreferrer" target="_blank">@internationalized/date</a> package to handle dates.

If you're looking for a range calendar, check out the <a href="https://www.shadcn-svelte.com/docs/components/range-calendar" class="font-medium underline underline-offset-4">Range Calendar</a> component.

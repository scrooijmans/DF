# JavaScript Data GridSparklines - Column Customisation

<span class="image placeholder _frameworkLogo_192zo_1" original-image-src="https://www.ag-grid.com/_astro/javascript.DOqolN0O.svg" original-image-title="">javascript logo</span>JavaScript<img src="out_ag_grid/sparklines-column-customisation/index_media/ebb52304aa2fe75a94cd1555a353273f67162700.svg" class="lucide lucide-chevron-down _chevronDown_6498a_108" />

Enterprise<img src="out_ag_grid/sparklines-column-customisation/index_media/7e74cacfc7ea261f2c66286804408c45e78f84a2.svg" class="_icon_1jqgc_1 icon" />

This section shows how Column Sparklines can be customised by overriding the default column options.

The following can be used to customise Column Sparklines:

- [Column Fill Options](https://www.ag-grid.com/javascript-data-grid/sparklines-column-customisation/#column-fill-options)
- [Column Stroke Options](https://www.ag-grid.com/javascript-data-grid/sparklines-column-customisation/#column-stroke-options)
- [Column Padding Options](https://www.ag-grid.com/javascript-data-grid/sparklines-column-customisation/#column-padding-options)
- [Column Label Options](https://www.ag-grid.com/javascript-data-grid/sparklines-column-customisation/#column-label-options)
- [Axis Line Options](https://www.ag-grid.com/javascript-data-grid/sparklines-column-customisation/#axis-line-options)
- [Sparkline Padding Options](https://www.ag-grid.com/javascript-data-grid/sparklines-column-customisation/#sparkline-padding-options)

Also see [Additional Customisations](https://www.ag-grid.com/javascript-data-grid/sparklines-column-customisation/#additional-customisations) for more advanced customisations that are common across all sparklines.

The snippet below shows option overrides for the Column Sparkline:

``` code
sparklineOptions: {
    type: 'bar',
    direction: 'vertical',
    fill: '#91cc75',
    stroke: '#91cc75',
    highlight: {
        highlightedItem: {
            fill: 'orange'
        }
    },
    axis: {
        type: 'category',
        paddingInner: 0.3,
        paddingOuter: 0.1,
    }
},
```

The following example demonstrates the results of the Column Sparkline options above:

<img src="out_ag_grid/sparklines-column-customisation/index_media/8ded47c1186e17f7830cf2d4518c3630384cfc60.svg" class="logomark" />

## Column Fill Options <a href="https://www.ag-grid.com/javascript-data-grid/sparklines-column-customisation/#column-fill-options" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/sparklines-column-customisation/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

To apply a custom color to the columns, set the `fill` property in `sparklineOptions` as shown:

``` code
sparklineOptions: {
    type: 'bar',
    direction: 'vertical',
    fill: '#91cc75', // sets the column fill
}
```

<span id="image-resources/default.png" class="image placeholder _image_vq8h5_1" original-image-src="https://www.ag-grid.com/_astro/default.D_aj5MVN.png" original-image-title="">Column fill default</span>

Custom fill

It is possible to set the fill for the highlighted state of the column by adding `fill` in `highlight` options as follows:

``` code
sparklineOptions: {
    type: 'bar',
    direction: 'vertical',
    highlight: {
        highlightedItem: {
            fill: 'orange', // sets the highlighted column fill
        }
    }
}
```

<span id="image-resources/default-highlighted.png" class="image placeholder _image_vq8h5_1" original-image-src="https://www.ag-grid.com/_astro/default-highlighted.C8u_M2RN.png" original-image-title="">Highlighted Column fill default</span>

Default highlighted fill

<span id="image-resources/custom-highlighted-fill.png" class="image placeholder _image_vq8h5_1" original-image-src="https://www.ag-grid.com/_astro/custom-highlighted-fill.9uxDAWo2.png" original-image-title="">Highlighted Column fill customisation</span>

Custom highlighted fill

The given `fill` string can be in one of the following formats:

- `#rgb` - Short Hex Code
- `#rrggbb` - Hex Code
- `rgb(r, g, b)` - RGB
- `rgba(r, g, b, a)` - RGB with an alpha channel
- CSS color keyword - such as `aqua`, `orange`, etc.

## Column Stroke Options <a href="https://www.ag-grid.com/javascript-data-grid/sparklines-column-customisation/#column-stroke-options" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/sparklines-column-customisation/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

By default, the `strokeWidth` of each column is `0`, so no outline is visible around the columns.

To add a stroke, modify the `strokeWidth` and `stroke` properties as shown below.

``` code
sparklineOptions: {
    type: 'bar',
    direction: 'vertical',
    stroke: '#ec7c7d', // sets the column stroke
    strokeWidth: 2, // sets the column stroke width
    highlight: {
        highlightedItem: {
            stroke: '#b5ec7c', // sets the highlighted column stroke
            strokeWidth: 2, // sets the highlighted column stroke width
        }
    }
}
```

- In the snippet above, we have configured the column stroke to be `2`px in the un-highlighted state, and `2`px in the highlighted state.
- Note that the `stroke` property is also different depending on the highlighted state of the column.

## Column Padding Options <a href="https://www.ag-grid.com/javascript-data-grid/sparklines-column-customisation/#column-padding-options" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/sparklines-column-customisation/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

The spacing between columns is adjustable via the `paddingInner` property. This property takes values between 0 and 1 and is available for axis `type: 'category'` only.

It is a proportion of the step?, which is the interval between the start of a band and the start of the next band.

Here's an example.

``` code
sparklineOptions: {
    type: 'bar',
    direction: 'vertical',
    axis: {
        type: 'category',
        paddingInner: 0.5, // sets the padding between columns.
    }
}
```

<span id="image-resources/default.png" class="image placeholder _image_vq8h5_1" original-image-src="https://www.ag-grid.com/_astro/default.D_aj5MVN.png" original-image-title="">Column padding default</span>

Default

<span id="image-resources/custom-padding-inner.png" class="image placeholder _image_vq8h5_1" original-image-src="https://www.ag-grid.com/_astro/custom-padding-inner.DZLFOA9E.png" original-image-title="">PaddingInner customisation</span>

Custom paddingInner

The padding on the outer edges of the first and last columns can also be adjusted. As with `paddingInner`, this value can be between 0 and 1 and is available for axis `type: 'category'` only.

If the value of `paddingOuter` is increased, the axis line will stick out more at both ends of the sparkline.

Here's a snippet where the `paddingOuter` is set to `0`.

``` code
sparklineOptions: {
    type: 'bar',
    direction: 'vertical',
    axis: {
        type: 'category',
        paddingOuter: 0, // sets the padding on the outer edge of the first and last columns.
    }
}
```

In this case there will be no gap on either end of the sparkline, i.e. between the axis line start and the first column and the axis line end and the last column. This is demonstrated below in the middle sparkline.

## Column Label Options <a href="https://www.ag-grid.com/javascript-data-grid/sparklines-column-customisation/#column-label-options" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/sparklines-column-customisation/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

To enable column labels, set the `enabled` property in `label` options as shown:

``` code
sparklineOptions: {
    type: 'bar',
    direction: 'vertical',
    label: {
        enabled: true // show column labels
    }
}
```

<span id="image-resources/default.png" class="image placeholder _image_vq8h5_1" original-image-src="https://www.ag-grid.com/_astro/default.D_aj5MVN.png" original-image-title="">Column default</span>

Default

<span id="image-resources/default-label.png" class="image placeholder _image_vq8h5_1" original-image-src="https://www.ag-grid.com/_astro/default-label.7V561Gg2.png" original-image-title="">Column labels enabled</span>

Label enabled

<img src="out_ag_grid/sparklines-column-customisation/index_media/8ded47c1186e17f7830cf2d4518c3630384cfc60.svg" class="logomark" />

It is possible to change the text value displayed as the label of individual columns by adding a `formatter` callback function to `label` options as follows:

``` code
sparklineOptions: {
    type: 'bar',
    direction: 'vertical',
    label: {
        enabled: true,
        formatter: labelFormatter
    }
}

function labelFormatter({ value }) {
    return `${value}%`
}
```

<span id="image-resources/default.png" class="image placeholder _image_vq8h5_1" original-image-src="https://www.ag-grid.com/_astro/default.D_aj5MVN.png" original-image-title="">Column default</span>

Custom label text

To customise the label text style, set the style attributes in `label` options as follows:

``` code
sparklineOptions: {
    type: 'bar',
    direction: 'vertical',
    label: {
        enabled: true,
        fontWeight: 'bold',
        fontStyle: 'italic',
        fontSize: 9,
        fontFamily: 'Arial, Helvetica, sans-serif',
        color: 'black',
    }
}
```

Custom label text styles

The position of the labels can be specified by setting the `placement` property in `label` options. By default, the labels are positioned at the end of the columns on the inside, i.e. `placement` is set to `insideEnd `. The snippet below shows how the positioning of the label can be modified:

``` code
sparklineOptions: {
    type: 'bar',
    direction: 'vertical',
    label: {
        enabled: true,
        placement: 'inside-center', // positions the labels in the center of the columns
    }
}
```

Label `placement` options include `inside-center`, `inside-start`, `inside-end`, `outside-start` or `outside-end` from the [AG Charts Label Placement](https://www.ag-grid.com/charts/javascript/bar-series/#reference-AgBarSeriesOptions-label-placement) documentation. ## Sparkline Padding Options <a href="https://www.ag-grid.com/javascript-data-grid/sparklines-column-customisation/#sparkline-padding-options" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/sparklines-column-customisation/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

To add extra space around the sparklines, custom `padding` options can be applied in the following way.

``` code
sparklineOptions: {
    type: 'bar',
    direction: 'vertical',
    // sets the padding around the sparklines
    padding: {
        top: 10,
        right: 5,
        bottom: 10,
        left: 5
    },
}
```

- The `top`, `right`, `bottom` and `left` properties are all optional and can be modified independently.

# JavaScript Data GridFill Handle

<span class="image placeholder _frameworkLogo_192zo_1" original-image-src="https://www.ag-grid.com/_astro/javascript.DOqolN0O.svg" original-image-title="">javascript logo</span>JavaScript<img src="out_ag_grid/cell-selection-fill-handle/index_media/ebb52304aa2fe75a94cd1555a353273f67162700.svg" class="lucide lucide-chevron-down _chevronDown_6498a_108" />

Enterprise<img src="out_ag_grid/cell-selection-fill-handle/index_media/7e74cacfc7ea261f2c66286804408c45e78f84a2.svg" class="_icon_1jqgc_1 icon" />

When working with cell selection, a Fill Handle allows you to run operations on cells as you adjust the size of the range.

## Enabling the Fill Handle <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection-fill-handle/#enabling-the-fill-handle" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection-fill-handle/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

To enable the Fill Handle, set `cellSelection.handle` to `{ mode: 'fill' }` in the `gridOptions` as shown below:

``` code
const gridOptions = {
    cellSelection: {
        handle: {
            mode: 'fill',
        }
    },

    // other grid options ...
}
```

The example below demonstrates the [default behaviour](https://www.ag-grid.com/javascript-data-grid/cell-selection-fill-handle/#default-fill-handle) with the minimal configuration above:

<img src="out_ag_grid/cell-selection-fill-handle/index_media/8ded47c1186e17f7830cf2d4518c3630384cfc60.svg" class="logomark" />

## Default Fill Handle <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection-fill-handle/#default-fill-handle" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection-fill-handle/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

The default Fill Handle behaviour will be as close as possible to other spreadsheet applications. Note the following:

### Single Cell <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection-fill-handle/#single-cell" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection-fill-handle/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

- When a single cell is selected and the range is increased, the value of that cell will be copied to the cells added to the range.
- When a single cell containing a **number** value is selected and the range is increased while pressing the  Alt key, that value will be incremented (or decremented if dragging to the left or up) by the value of one until all new cells have been filled.

### Multi Cell <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection-fill-handle/#multi-cell" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection-fill-handle/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

- When a range of numbers is selected and that range is extended, the Grid will detect the linear progression of the selected items and fill the extra cells with calculated values.
- When a range of strings or a mix of strings and numbers are selected and that range is extended, the range items will be copied in order until all new cells have been properly filled.
- When a range of numbers is selected and the range is increased while pressing the  Alt key, the behaviour will be the same as when a range of strings or mixed values is selected.

### Range Reduction <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection-fill-handle/#range-reduction" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection-fill-handle/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

- When reducing the size of the range, cells that are no longer part of the range will be cleared (set to `null`). If your column uses a `valueParser`, it will receive an empty string (`''`) as the new value.

<img src="out_ag_grid/cell-selection-fill-handle/index_media/8ded47c1186e17f7830cf2d4518c3630384cfc60.svg" class="logomark" />

### Preventing Range Reduction <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection-fill-handle/#preventing-range-reduction" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection-fill-handle/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

Reducing a range selection with the Fill Handle will clear cell contents by default, as can be observed in the [cell reduction](https://www.ag-grid.com/javascript-data-grid/cell-selection-fill-handle/#range-reduction) example above.

If this behaviour for decreasing selection needs to be prevented, the flag `cellSelection.handle.suppressClearOnFillReduction` should be set to `true`.

<img src="out_ag_grid/cell-selection-fill-handle/index_media/8ded47c1186e17f7830cf2d4518c3630384cfc60.svg" class="logomark" />

## Fill Handle Axis <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection-fill-handle/#fill-handle-axis" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection-fill-handle/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

By default, the Fill Handle can be dragged horizontally or vertically. If you wish to restrict the permitted direction of dragging to either horizontal or vertical, set `cellSelection.handle.direction` to either `x` for horizontal or `y` for vertical.

``` code
const gridOptions = {
    cellSelection: {
        handle: {
            mode: 'fill',
            direction: 'x', // Fill Handle can only be dragged horizontally
        }
    },

    // other grid options ...
}
```

<img src="out_ag_grid/cell-selection-fill-handle/index_media/8ded47c1186e17f7830cf2d4518c3630384cfc60.svg" class="logomark" />

## Double-Click Fill <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection-fill-handle/#double-click-fill" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection-fill-handle/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

When the fill handle direction is `'y'` or `'xy'`, double-clicking on the fill handle will perform a fill operation on all cells below the selected cells. Similarly, when the fill handle direction is `'x'`, double-clicking on the fill handle will perform a fill operation on all cells to the right of the selected cells.

This is enabled by default when the fill handle is enabled and does not require separate configuration.

## Custom User Function <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection-fill-handle/#custom-user-function" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection-fill-handle/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

Often there is a need to use a custom method to fill values instead of simply copying values or increasing number values using linear progression. In these scenarios, the `cellSelection.handle.setFillValue` callback should be used.

``` code
const gridOptions = {
    columnDefs: [
        { field: 'country' },
        { field: 'year' },
        { field: 'sport' },
        { field: 'total' }
    ],
    cellSelection: {
        handle: {
            mode: 'fill',
            setFillValue: (params) => 'Custom Fill Value',
        }
    },

    // other grid options ...
}
```

### Skipping Columns in the Fill Operation <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection-fill-handle/#skipping-columns-in-the-fill-operation" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection-fill-handle/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

## Read Only Edit <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection-fill-handle/#read-only-edit" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection-fill-handle/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

When the grid is in [Read Only Edit](https://www.ag-grid.com/javascript-data-grid/value-setters/#read-only-edit) mode the Fill Handle will not update the data inside the grid. Instead the grid fires `cellEditRequest` events allowing the application to process the update request.

# JavaScript Data GridSparklines - Points of Interest

<span class="image placeholder _frameworkLogo_192zo_1" original-image-src="https://www.ag-grid.com/_astro/javascript.DOqolN0O.svg" original-image-title="">javascript logo</span>JavaScript<img src="out_ag_grid/sparklines-points-of-interest/index_media/ebb52304aa2fe75a94cd1555a353273f67162700.svg" class="lucide lucide-chevron-down _chevronDown_6498a_108" />

This section covers customisation of Sparkline Points of Interest.

Some data points in the sparklines are special and can be emphasised to allow for quick identification and comparisons across the values of a single sparkline or across multiple sparklines of the same type. These include:

- First and Last
- Minimum and Maximum
- Positive and Negative

First and last

<span id="image-resources/area-sparkline.png" class="image placeholder _image_vq8h5_1" original-image-src="https://www.ag-grid.com/_astro/area-sparkline.C1mIivvz.png" original-image-title="">Area sparkline</span>

Minimum and Maximum

Negative and positive

These special points can be customised via the `styler` callback function to make them stand out from the rest of the data points which are using the global styles.

- The styler is a callback function used to return formatting for individual data points based on the given parameters.
- The styler receives an input parameter according to the sparkline type.

## Line and Area Sparklines Points of Interest <a href="https://www.ag-grid.com/javascript-data-grid/sparklines-points-of-interest/#line-and-area-sparklines-points-of-interest" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/sparklines-points-of-interest/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

In the line and area sparklines, each data point is represented by a marker. To customise the points of interest, the `styler` function is added to the `marker` options:

``` code
sparklineOptions: {
    marker: {
        enabled: true,
        itemStyler: (params: AgSeriesMarkerStylerParams): AgSeriesMarkerStyle => ..., // add itemStyler to marker options
    },
}
```

The `itemStyler` callback function will receive an input parameter of type [`AgSeriesMarkerStylerParams`](https://www.ag-grid.com/javascript-data-grid/sparklines-api-sparkline-options/#sparkline-item-styler).

The function return type should be [`AgSeriesMarkerStyle`](https://www.ag-grid.com/javascript-data-grid/sparklines-api-sparkline-options/#sparkline-item-styler), allowing the following attributes to be customised:

- size
- fill
- stroke
- strokeWidth

The following sections outline how the attributes mentioned above can be customised for various special points of interest:

- [First and Last](https://www.ag-grid.com/javascript-data-grid/sparklines-points-of-interest/#first-and-last)
- [Minimum and Maximum](https://www.ag-grid.com/javascript-data-grid/sparklines-points-of-interest/#min-and-max)
- [Positive and Negative](https://www.ag-grid.com/javascript-data-grid/sparklines-points-of-interest/#positive-and-negative)

### First and Last <a href="https://www.ag-grid.com/javascript-data-grid/sparklines-points-of-interest/#first-and-last" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/sparklines-points-of-interest/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

Let's say we have a line sparkline where the markers are all `'skyblue'` but we want to make the first and last markers stand out with a purple `fill` and `stroke` style.

We can do this by adding the following styler to the `marker` options.

``` code
const itemStyler = (params: AgSeriesMarkerStylerParams): AgSeriesMarkerStyle => {
    const { first, last } = params;

    return {
        size: first || last ? 5 : 3,
        fill: first || last ? '#9a60b4' : 'skyblue',
        stroke: first || last ? '#9a60b4' : 'skyblue'
    }
}
```

- In the snippet above, `first` and `last` boolean values are extracted from the params object and used to conditionally set the `size`, `fill` and `stroke` of the markers.
- If the given data point is the first or last point i.e. if `first` or `last` is `true`, the `size` of the marker is set to `5`px. All other markers will be `3`px.
- Similar conditional logic is applied to colorise the markers to distinguish the first and last points from the rest.

### Min and Max <a href="https://www.ag-grid.com/javascript-data-grid/sparklines-points-of-interest/#min-and-max" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/sparklines-points-of-interest/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

Similar to first and last, to emphasise the min and max data points, the `min` and `max` booleans from the styler params can be used to conditionally style the markers.

``` code
const itemStyler = (params: AgSeriesMarkerStylerParams): AgSeriesMarkerStyle => {
    const { min, max } = params;

    return {
        size: min || max ? 5 : 3,
        fill: min ? '#ee6666' : max ? '#3ba272' : 'skyBlue',
        stroke: min ? '#ee6666' : max ? '#3ba272' : 'skyBlue',
    }
}
```

- If the data point is a minimum or a maximum point  if `min` or `max` is `true`  the size is set to `5`px, otherwise it is set to`3`px.
- If the marker represents a minimum point, the `fill` and `stroke` are set to red, if the marker represents a maximum point, the `fill` and `stroke` are set to green. Otherwise the fill and stroke are set to sky blue.

### Positive and Negative <a href="https://www.ag-grid.com/javascript-data-grid/sparklines-points-of-interest/#positive-and-negative" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/sparklines-points-of-interest/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

The positive and negative values can be distinguished by adding a `styler` which returns styles based on the `yValue` of the data point.

This is demonstrated in the snippet below.

``` code
const itemStyler = (params: AgSeriesMarkerStylerParams): AgSeriesMarkerStyle => {
    const { yValue } = params;

    return {
        // if yValue is negative, the marker should be 'red', otherwise it should be 'green'
        fill: yValue < 0 ? 'red' : 'green',
        stroke: yValue < 0 ? 'red' : 'green'
    }
}
```

## Column And Bar Sparklines Points of Interest <a href="https://www.ag-grid.com/javascript-data-grid/sparklines-points-of-interest/#column-and-bar-sparklines-points-of-interest" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/sparklines-points-of-interest/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

Bar sparklines are just transposed column sparklines and have the same configuration. This section only covers column sparkline examples but the same applies for bar sparklines.

In the column sparklines, each data point is represented by a rectangle/ column. The `styler` callback function applies to the individual columns and is added to `sparklineOptions`:

``` code
sparklineOptions: {
    type: 'bar',
    direction: 'vertical',
    itemStyler: columnFormatter, // add styler to sparklineOptions
}
```

The `itemStyler` will receive an input parameter with values associated with the data point it represents. The input parameter type is [`columnFormatterParams`](https://www.ag-grid.com/javascript-data-grid/).

The function return type should be [`ItemStylerFormat`](https://www.ag-grid.com/javascript-data-grid/), allowing these attributes to be customised:

- fill
- stroke
- strokeWidth

The following sections outline how the attributes mentioned above can be customised for various special points of interest:

- [First and Last](https://www.ag-grid.com/javascript-data-grid/sparklines-points-of-interest/#first-and-last-1)
- [Minimum and Maximum](https://www.ag-grid.com/javascript-data-grid/sparklines-points-of-interest/#min-and-max-1)
- [Positive and Negative](https://www.ag-grid.com/javascript-data-grid/sparklines-points-of-interest/#positive-and-negative-1)

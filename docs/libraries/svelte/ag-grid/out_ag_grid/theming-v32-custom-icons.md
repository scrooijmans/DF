# JavaScript Data GridLegacy Themes: Custom Icons

<span class="image placeholder _frameworkLogo_192zo_1" original-image-src="https://www.ag-grid.com/_astro/javascript.DOqolN0O.svg" original-image-title="">javascript logo</span>JavaScript<img src="out_ag_grid/theming-v32-custom-icons/index_media/ebb52304aa2fe75a94cd1555a353273f67162700.svg" class="lucide lucide-chevron-down _chevronDown_6498a_108" />

This section details how to provide your own icons for the grid and style grid icons for your application requirements

<img src="out_ag_grid/theming-v32-custom-icons/index_media/ae2971b6b482d433157d4aac53e206fe28cf84e4.svg" class="_icon_1jqgc_1 icon" />

This page describes the grid's legacy theming system that was the default in v32 and before, for the benefit of applications that have not yet migrated to the Theming API. These themes are deprecated and will be removed in a future major version. You may want to visit the [new theming docs](https://www.ag-grid.com/javascript-data-grid/custom-icons/) or check out the [migration guide](https://www.ag-grid.com/javascript-data-grid/theming-migration/).

## Swapping the Provided Icon Fonts <a href="https://www.ag-grid.com/javascript-data-grid/theming-v32-custom-icons/#swapping-the-provided-icon-fonts" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/theming-v32-custom-icons/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

Each provided theme comes with its own icon font. It is simple to use one provided theme with another theme's icons. Set `--ag-icon-font-family` to one of: `agGridQuartz`, `agGridAlpine`, `agGridBalham` or `agGridMaterial`. You can compare the available icon fonts in the [Provided Icons list](https://www.ag-grid.com/javascript-data-grid/theming-v32-custom-icons/#provided-icons).

If you are using the Sass API, it will embed the required font data for you. Apps using CSS should load the font's CSS file from whatever location they are loading `ag-grid.css` and other CSS files. For example to use the Material icons in the Quartz theme:

1.  Load `agGridMaterialFont.css`
2.  (optional) switch `ag-theme-quartz.css` for `ag-theme-quartz-no-font.css` to save a few kB if you no longer require the Quartz icons
3.  Set the CSS variable `--ag-icon-font-family: agGridMaterial`

## Using Alternative Icons <a href="https://www.ag-grid.com/javascript-data-grid/theming-v32-custom-icons/#using-alternative-icons" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/theming-v32-custom-icons/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

Grid icons can use an icon font or images. The grid exposes a number of CSS variables to control the icons. Each variable can optionally be suffixed with the name of a specific icon, and will affect only that icon, e.g. setting `--ag-icon-font-color-loading` sets the color of the "loading" icon.

Icon font variables:

- `--ag-icon-font-family[-icon-name]` set the icon font family.
- `--ag-icon-font-code[-icon-name]` set the unicode character of the icon within the icon font, using a unicode escape sequence like `"\f247"`.
- `--ag-icon-font-weight[-icon-name]` sets the font weight, used for icon fonts that require a specific weight to work.
- `--ag-icon-font-color[-icon-name]` set the color of the icon.
- `--ag-icon-font-display[-icon-name]` set this to `none` to hide the icon font if you want to provide an image-based icon instead.

Icon image variables:

- `--ag-icon-image[-icon-name]` set this to `url(/path/to/icon.svg)`
- `--ag-icon-image-opacity[-icon-name]` set the opacity of an icon. Defaults to 0.9 if not set.
- `--ag-icon-image-display[-icon-name]` set this to `none` to hide the icon image if you want to provide an font-based icon instead.

Other icon variables:

- `--ag-icon-size` sets the height and width of icons in pixels.

### Example: using an alternative icon font <a href="https://www.ag-grid.com/javascript-data-grid/theming-v32-custom-icons/#example-using-an-alternative-icon-font" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/theming-v32-custom-icons/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

This example demonstrates globally changing the icon font, and also selectively replacing individual icons:

``` code
.ag-theme-quartz {
  /* replace all icons in the grid with icons from Font Awesome */
  --ag-icon-font-family: "Font Awesome 5 Free";
  --ag-icon-font-code-aggregation: "\f247";
  --ag-icon-font-code-arrows: "\f0b2";
  --ag-icon-font-code-asc: "\f062";
  /* ... and so on - because --ag-icon-font-family sets the font for
     all icons, a new code must be provided for every one. If you only
     wanted to replace some icons, use --ag-icon-font-family-{icon-name} */

  /* selectively replace the the group icon with one from Material Design Icons */
  --ag-icon-font-family-group: "Material Design Icons";
  --ag-icon-font-code-group: "\F0328";
  --ag-icon-font-color-group: red;
  --ag-icon-font-weight-group: normal;
}
```

### Example: SVG icons <a href="https://www.ag-grid.com/javascript-data-grid/theming-v32-custom-icons/#example-svg-icons" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/theming-v32-custom-icons/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

``` code
.ag-theme-quartz {

     icons, an image must be provided for every one. If you only wanted
     to replace some icons, use --ag-icon-font-display-{icon-name} */
}
```

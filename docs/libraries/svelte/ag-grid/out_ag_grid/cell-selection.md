# JavaScript Data GridCell Selection

<span class="image placeholder _frameworkLogo_192zo_1" original-image-src="https://www.ag-grid.com/_astro/javascript.DOqolN0O.svg" original-image-title="">javascript logo</span>JavaScript<img src="out_ag_grid/cell-selection/index_media/ebb52304aa2fe75a94cd1555a353273f67162700.svg" class="lucide lucide-chevron-down _chevronDown_6498a_108" />

Enterprise<img src="out_ag_grid/cell-selection/index_media/7e74cacfc7ea261f2c66286804408c45e78f84a2.svg" class="_icon_1jqgc_1 icon" />

Cell selection allows Excel-like selection of ranges of cells. Cell selections are useful for visually highlighting data, copying data to the [Clipboard](https://www.ag-grid.com/javascript-data-grid/clipboard/), or for doing aggregations using the [Status Bar](https://www.ag-grid.com/javascript-data-grid/status-bar/).

## Enabling Cell Selection <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection/#enabling-cell-selection" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

Cell Selection is enabled by setting the `gridOptions.cellSelection` to `true`, or to a configuration object.

``` code
const gridOptions = {
    cellSelection: true,

    // other grid options ...
}
```

When enabled, ranges can be selected in the following ways:

- **Mouse Drag:** Click the mouse down on a cell and drag and release the mouse over another cell. A range will be created between the two cells and clear any existing ranges.

- **Ctrl & Mouse Drag:** Holding ^ Ctrl key while creating a range using mouse drag **outside an existing range** will create a new cell range selection and keep any existing ranges.

- **Shift & Click:** Clicking on one cell to focus that cell, then holding down  Shift while clicking another cell, will create a range between both cells.

- **Shift & Arrow Keys:** Focusing a cell and then holding down  Shift and using the arrow keys will create a range starting from the focused cell.

- **Ctrl & Shift & Arrow Keys:** Focusing a cell and then holding down ^ Ctrl +  Shift and using the arrow keys will create a range starting from the focused cell to the last cell in the direction of the Arrow pressed.

### Cell Range Deselection <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection/#cell-range-deselection" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

It is possible to deselect part of existing ranges in the following ways:

- **Ctrl & Mouse Drag:** Holding ^ Ctrl and dragging a range starting **within an existing range** will cause any cells covered by the new range to be deselected.

- **Ctrl & Click:** Holding ^ Ctrl and clicking a cell will deselect just that cell.

Note that deselecting part of a range can split the range into multiple ranges, since individual ranges have the limitation of being rectangular.

The example below demonstrates simple cell selection. Cell ranges can be selected in all the ways described above.

## Prevent Selection of Multiple Ranges <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection/#prevent-selection-of-multiple-ranges" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

By default multiple ranges can be selected. To restrict cell selection to a single range, even if the ^ Ctrl key is held down, set `cellSelection.suppressMultiRanges` to `true`.

``` code
const gridOptions = {
    cellSelection: {
        suppressMultiRanges: true,
    },

    // other grid options ...
}
```

## Ranges with Pinned Columns and Rows <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection/#ranges-with-pinned-columns-and-rows" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

It is possible to select a cell range that spans pinned and non-pinned sections of the grid. If you do this, the selected range will not have any gaps with regards to the column or row order.

For example, if you start the drag on the left pinned area and drag to the right pinned area, then all of the columns in the centre area will also be part of the range. Likewise with pinned rows, no row gaps will occur if a cell range spans into pinned rows. A range will be continuous between the rows pinned to the top, the centre, and the rows pinned to the bottom.

This can be thought of as follows: if you have a grid with pinned rows and / or columns, then 'flatten out' the grid in your head so that all rows and columns are visible, then the cell selection will work as you would expect in the flattened out version where only full rectangles can be selectable.

## Highlight Headers <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection/#highlight-headers" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

It is possible to highlight the Grid Headers that are part of a range by setting the `cellSelection.enableHeaderHighlight` to `true`.

``` code
const gridOptions = {
    cellSelection: {
        enableHeaderHighlight: true,
    },

    // other grid options ...
}
```

## Copy Cell Range Down <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection/#copy-cell-range-down" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

When you have more than one row selected in a range, pressing keys ^ Ctrl+D will copy the top row values to all other rows in the selected range.

By default, the Value Formatter and Value Parser will be used whilst copying the range via the [Use Value Formatter For Export](https://www.ag-grid.com/javascript-data-grid/value-formatters/#formatting-for-export) and [Use Value Parser for Import](https://www.ag-grid.com/javascript-data-grid/value-parsers/#use-value-parser-for-import) features.

## Bulk Cell Edit <a href="https://www.ag-grid.com/javascript-data-grid/cell-selection/#bulk-cell-edit" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/cell-selection/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

When [Cell Editing](https://www.ag-grid.com/javascript-data-grid/cell-editing/) is enabled, and a cell range is selected, typing to enter a new value and pressing the ^ Ctrl+ Enter keys will set the newly provided cell value to all the editable cells in the selected cell range.

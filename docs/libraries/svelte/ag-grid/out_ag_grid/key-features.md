### Mapping Values <a href="https://www.ag-grid.com/javascript-data-grid/key-features/#mapping-values" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/key-features/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

The `field` or `valueGetter` attributes [Map Data to Columns](https://www.ag-grid.com/javascript-data-grid/column-definitions/). A field maps to a field in the data. A [Value Getter](https://www.ag-grid.com/javascript-data-grid/value-getters/) is a function callback that returns the cell value.

The `headerName` provides the title for the header. If missing the title is derived from `field`.

``` code
const gridOptions = {
    columnDefs: [
        { headerName: "Make & Model", valueGetter: p => p.make + ' ' + p.model},
        { field: "price" },
    ],

    // other grid options ...
}
```

### Text Formatting <a href="https://www.ag-grid.com/javascript-data-grid/key-features/#text-formatting" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/key-features/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

Format text for cell content using a [Value Formatter](https://www.ag-grid.com/javascript-data-grid/value-formatters/).

``` code
const gridOptions = {
    columnDefs: [
        { field: "price", valueFormatter: p => '' + p.value.toLocaleString() },
    ],

    // other grid options ...
}
```

### Cell Components <a href="https://www.ag-grid.com/javascript-data-grid/key-features/#cell-components" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/key-features/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

Add buttons, checkboxes or images to cells with a [Cell Component](https://www.ag-grid.com/javascript-data-grid/component-cell-renderer/).

``` code
class CustomButtonComponent {
    eGui;
    eButton;
    eventListener;

    init(params) {
        this.eGui = document.createElement("div");
        let button = document.createElement("button");
        button.className = "btn-simple";
        button.textContent = "Push Me!";
        this.eventListener = () => alert("clicked");
        button.addEventListener("click", this.eventListener);
        this.eGui.appendChild(button);
    }

    getGui() {
        return this.eGui;
    }

    refresh(params) {
        return true;
    }

    destroy() {
        if (button) {
            button.removeEventListener("click", this.eventListener);
        }
    }
}


const columnDefs = [
    { field: "button", cellRenderer: CustomButtonComponent },
    // ...
];
```

``` code
const gridOptions = {

        { field: "make", flex: 2 }, //This column will be twice as wide as the others
        { field: "model", flex: 1 },
        { field: "price", flex: 1 },

    // other grid options ...
}
```

## Working with Data <a href="https://www.ag-grid.com/javascript-data-grid/key-features/#working-with-data" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/key-features/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

By default, the row data is used to infer the [Cell Data Type](https://www.ag-grid.com/javascript-data-grid/cell-data-types/). The cell data type allows grid features, such as filtering and editing, to work without additional configuration.

### Filtering <a href="https://www.ag-grid.com/javascript-data-grid/key-features/#filtering" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/key-features/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

[Column Filters](https://www.ag-grid.com/javascript-data-grid/filtering/) are embedded into each column menu. These are enabled using the `filter` attribute. The filter type is inferred from the cell data type.

``` code
const gridOptions = {
    columnDefs: [
        { field: "make", filter: true },
    ],

    // other grid options ...
}
```

There are 5 [Provided Filters](https://www.ag-grid.com/javascript-data-grid/filtering/) which can be set through this attribute. You can also create your own [Custom Filter](https://www.ag-grid.com/javascript-data-grid/component-filter/).

[Floating Filters](https://www.ag-grid.com/javascript-data-grid/floating-filters/) embed the Column Filter into the header for ease of access.

``` code
const gridOptions = {
    columnDefs: [
        { field: "make", filter: true, floatingFilter: true },
    ],

    // other grid options ...
}
```

### Editing <a href="https://www.ag-grid.com/javascript-data-grid/key-features/#editing" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/key-features/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

Enable [Editing](https://www.ag-grid.com/javascript-data-grid/cell-editing/) by setting the `editable` attribute to `true`. The cell editor is inferred from the cell data type.

``` code
const gridOptions = {
    columnDefs: [
        { field: "make", editable: true },
    ],

    // other grid options ...
}
```

Set the cell editor type using the `cellEditor` attribute. There are 7 [Provided Cell Editors](https://www.ag-grid.com/javascript-data-grid/provided-cell-editors/) which can be set through this attribute. You can also create your own [Custom Editors](https://www.ag-grid.com/javascript-data-grid/cell-editors/).

``` code
const gridOptions = {
    columnDefs: [
        {
            field: "make",
            editable: true,
            cellEditor: 'agSelectCellEditor',
            cellEditorParams: {
                values: ['Tesla', 'Ford', 'Toyota'],
            },
        },
    ],

    // other grid options ...
}
```

## Themes & Style <a href="https://www.ag-grid.com/javascript-data-grid/key-features/#themes--style" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/key-features/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

### Themes <a href="https://www.ag-grid.com/javascript-data-grid/key-features/#themes" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/key-features/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

[Grid Themes](https://www.ag-grid.com/javascript-data-grid/theming/) define how the grid looks (colours, font, spacing etc). The default theme is called Quartz. You can choose a [different theme](https://www.ag-grid.com/javascript-data-grid/themes/), or customise a built-in theme by changing parameters. Here we create a new theme based on Quartz:

``` code
import { themeQuartz } from "ag-grid-community"; // or themeBalham, themeAlpine

const myTheme = themeQuartz.withParams({
    /* Low spacing = very compact */
    spacing: 2,
    /* Changes the colour of the grid text */
    foregroundColor: 'rgb(14, 68, 145)',
    /* Changes the colour of the grid background */
    backgroundColor: 'rgb(241, 247, 255)',
    /* Changes the header colour of the top row */
    headerBackgroundColor: 'rgb(228, 237, 250)',
    /* Changes the hover colour of the row*/
    rowHoverColor: 'rgb(216, 226, 255)',
});

const gridOptions = {
    theme: myTheme,
    ...
}
```

### Cell Style <a href="https://www.ag-grid.com/javascript-data-grid/key-features/#cell-style" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/key-features/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

Define rules to apply [Styling to Cells](https://www.ag-grid.com/javascript-data-grid/cell-styles/) using `cellClassRules`. This can be used, for example, to set cell background colour based on its value.

``` code
.rag-green {
    background-color: #33cc3344;
}
```

``` code
const gridOptions = {
    columnDefs: [{
        field: 'electric',
        cellClassRules: {
            // apply green to electric cars
            'rag-green': params => params.value === true,
        }
    }],

    // other grid options ...
}
```

Define rules to apply [Styling to Rows](https://www.ag-grid.com/javascript-data-grid/row-styles/) using `rowClassRules`. This allows changing style (e.g. row colour) based on row values.

``` code
.rag-red {
    background-color: #cc222244;
```

``` code
const gridOptions = {
    rowClassRules: {
        // apply red to Ford cars
        'rag-red': params => params.data.make === 'Ford',

}
```

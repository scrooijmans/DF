Learn how to upgrade AG Grid to a specific version.

These guides explain how to move from one version of AG Grid to another. They outline breaking changes, new features, and the adjustments you need to make in your code.

## Codemods <a href="https://www.ag-grid.com/javascript-data-grid/migration/#codemods" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/migration/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

All AG Grid releases from v31 onwards come with an accompanying Codemod to help automate the upgrade process. Codemods are scripts that fix the project's source files to address the majority of breaking changes and deprecations when upgrading from an older version.

Each of the migration guides listed below will include details on how to use Codemods to upgrade to that version. If you want to learn more about Codemods please review our dedicated [Codemods documentation](https://www.ag-grid.com/javascript-data-grid/codemods/).

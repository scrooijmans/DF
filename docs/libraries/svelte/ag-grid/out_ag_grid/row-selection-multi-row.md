# JavaScript Data GridMulti-Row Selection

<span class="image placeholder _frameworkLogo_192zo_1" original-image-src="https://www.ag-grid.com/_astro/javascript.DOqolN0O.svg" original-image-title="">javascript logo</span>JavaScript<img src="out_ag_grid/row-selection-multi-row/index_media/ebb52304aa2fe75a94cd1555a353273f67162700.svg" class="lucide lucide-chevron-down _chevronDown_6498a_108" />

Enable users to select many rows at once within a grid.

## Enabling Multi-Row Selection <a href="https://www.ag-grid.com/javascript-data-grid/row-selection-multi-row/#enabling-multi-row-selection" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/row-selection-multi-row/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

To enable multi-row selection set `rowSelection.mode` to `'multiRow'`:

``` code
const gridOptions = {
    rowSelection: { 
        mode: 'multiRow' 
    },

    // other grid options ...
}
```

The following example illustrates a basic multi-row selection configuration.

<img src="out_ag_grid/row-selection-multi-row/index_media/8ded47c1186e17f7830cf2d4518c3630384cfc60.svg" class="logomark" />

Click checkboxes to select or deselect a row. Alternatively, you can do this via the keyboard by focusing the row and pressing the ? Space key. Users can hold  Shift and then click a checkbox to add a range of adjacent rows to the selection.

Ranges of rows can be selected by holding down  Shift while clicking on checkboxes. This behaviour also applies when [Click Selection](https://www.ag-grid.com/javascript-data-grid/row-selection-multi-row/#enable-click-selection--deselection) is enabled, and in [Group Selection](https://www.ag-grid.com/javascript-data-grid/grouping-row-selection/).

## Removing Selection Checkboxes <a href="https://www.ag-grid.com/javascript-data-grid/row-selection-multi-row/#removing-selection-checkboxes" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/row-selection-multi-row/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

To prevent any row selection checkboxes from being rendered in rows, set `rowSelection.checkboxes` to `false`. To prevent the header checkbox from being rendered, set `rowSelection.headerCheckbox` to `false`. Setting both to `false` will disable the checkbox column. You will also need to enable click selection by setting `enableClickSelection: true`.

``` code
const gridOptions = {
    rowSelection: {
        mode: 'multiRow',
        checkboxes: false,
        headerCheckbox: false,
        enableClickSelection: true,
    },

    // other grid options ...
}
```

<img src="out_ag_grid/row-selection-multi-row/index_media/8ded47c1186e17f7830cf2d4518c3630384cfc60.svg" class="logomark" />

<img src="out_ag_grid/row-selection-multi-row/index_media/ae2971b6b482d433157d4aac53e206fe28cf84e4.svg" class="_icon_1jqgc_1 icon" />

You may pass a function to `rowSelection.checkboxes` to dynamically enable or disable checkboxes for given rows. For rows where both `isRowSelectable` and `rowSelection.checkboxes` return `false`, checkboxes will be hidden, rather than disabled.

## Configure Selectable Rows <a href="https://www.ag-grid.com/javascript-data-grid/row-selection-multi-row/#configure-selectable-rows" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/row-selection-multi-row/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

It is possible to specify which rows can be selected via the `rowSelection.isRowSelectable` callback function.

For instance if we only wanted to allow selection for rows where the 'year' property is less than 2007, we could implement the following:

``` code
const gridOptions = {
    rowSelection: {
        mode: 'multiRow',
        isRowSelectable: (rowNode) => rowNode.data ? rowNode.data.year < 2007 : false,
    },

    // other grid options ...
}
```

Rows for which `isRowSelectable` returns `false` cannot be selected at all, whether using the UI or the API.

<img src="out_ag_grid/row-selection-multi-row/index_media/8ded47c1186e17f7830cf2d4518c3630384cfc60.svg" class="logomark" />

Note that this example uses `hideDisabledCheckboxes` to hide disabled checkboxes, which can be toggled on or off.

## Selecting All Rows <a href="https://www.ag-grid.com/javascript-data-grid/row-selection-multi-row/#selecting-all-rows" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/row-selection-multi-row/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

All rows may be selected at once by using the header checkbox, which is enabled by default in `'multiRow'` mode.

The three possible values of `rowSelection.selectAll` are:

- `'all'`: *(Default)* Selecting the header checkbox selects all [selectable](https://www.ag-grid.com/javascript-data-grid/row-selection-multi-row/#configure-selectable-rows) rows in the grid.
- `'filtered'`: Selecting the header checkbox will select all rows that satisfy the currently active filter.
- `'currentPage'`: Selecting the header checkbox will select all rows that satisfy the currently active filter on the current page.

``` code
const gridOptions = {
    rowSelection: { 
        mode: 'multiRow',
        selectAll: 'filtered'
    },

    // other grid options ...
}
```

The example below demonstrates the three different modes available for `rowSelection.selectAll`:

<img src="out_ag_grid/row-selection-multi-row/index_media/8ded47c1186e17f7830cf2d4518c3630384cfc60.svg" class="logomark" />

Note that when `rowSelection.isRowSelectable` is defined, the header checkbox will only select selectable rows.

<img src="out_ag_grid/row-selection-multi-row/index_media/ae2971b6b482d433157d4aac53e206fe28cf84e4.svg" class="_icon_1jqgc_1 icon" />

The value of `rowSelection.selectAll` does not affect group selection behaviour, which is controlled by `rowSelection.groupSelects`. See [Row Grouping - Selecting Groups](https://www.ag-grid.com/javascript-data-grid/grouping-row-selection/) for more on this.

## Enable Click Selection & Deselection <a href="https://www.ag-grid.com/javascript-data-grid/row-selection-multi-row/#enable-click-selection--deselection" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/row-selection-multi-row/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

## Force Checkboxes to be Selected <a href="https://www.ag-grid.com/javascript-data-grid/row-selection-multi-row/#force-checkboxes-to-be-selected" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/row-selection-multi-row/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

It is possible to select a row via the Grid API and disable its checkbox to prevent users from deselecting it. This can be achieved by providing a function to `rowSelection.checkboxes`.

``` code
const gridOptions = {
    rowSelection: {
        mode: 'multiRow',
        checkboxes: (params) => params.data && params.data.year === 2012,
    },
    onFirstDataRendered: (params) => {
        const nodesToSelect = [];
        params.api.forEachNode((node) => {
            if (node.data && node.data.year <= 2008 && node.data.year >= 2004) {
                nodesToSelect.push(node);
            }
        });
        params.api.setNodesSelected({ nodes: nodesToSelect, newValue: true });
    },

    // other grid options ...
}
```

In the example below only rows with Year equal to 2012 can be selected or deselected by the user. Clicking the header checkbox, however, will select all rows even if their checkboxes are disabled.

<img src="out_ag_grid/row-selection-multi-row/index_media/8ded47c1186e17f7830cf2d4518c3630384cfc60.svg" class="logomark" />

## Selecting Multiple Rows without Ctrl key <a href="https://www.ag-grid.com/javascript-data-grid/row-selection-multi-row/#selecting-multiple-rows-without-ctrl-key" class="_linkIcon_152wk_1 _docsHeaderIcon_152wk_8 button-secondary" aria-label="Heading link">Copy Link<img src="out_ag_grid/row-selection-multi-row/index_media/76673052bb2634e0af37ce510b283b99cec64e64.svg" class="_icon_1jqgc_1 icon" /></a>

In certain circumstances, especially in the context of touchscreen devices, users may want to select multiple rows without having to use the ^ Ctrl key.

This can be accomplished by setting the `rowSelection.enableSelectionWithoutKeys` flag to `true`. You will also need to set `enableClickSelection` to `true`.

``` code
const gridOptions = {
    rowSelection: {
        mode: 'multiRow',
        checkboxes: false,
        headerCheckbox: false,
        enableSelectionWithoutKeys: true,
        enableClickSelection: true,
    },

    // other grid options ...
}
```

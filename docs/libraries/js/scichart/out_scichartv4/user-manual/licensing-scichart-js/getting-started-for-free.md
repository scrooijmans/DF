# Get Started with SciChart.js - Free Community & Commercial trial licensing

The below page will tell you how to get started with SciChart.js using our free community license, which works out of the box, without sign-up and can be used for commercial evaluation as well as non-commercial use.

![](out_scichartv4/user-manual/licensing-scichart-js/getting-started-for-free/index_media/f0173787a207cc0a720ca619ab3e16cfe886b3c7.svg)tip

Please be aware that starting usage of the SciChart software constitutes acceptance of the <a href="https://www.scichart.com/scichart-eula" rel="noopener noreferrer" target="_blank">licensing terms &amp; conditions</a>. SciChart is licensed per-developer with perpetual runtime, developer subscription with a free Community License for non-commercial use.

A helpful <a href="https://support.scichart.com/support/solutions/articles/101000385263-scichart-licensing-faq" rel="noopener noreferrer" target="_blank">licensing FAQ can be found here</a>.

## Is SciChart.js free?<a href="https://www.scichart.com/documentation/js/v4/user-manual/licensing-scichart-js/getting-started-for-free/#is-scichartjs-free" class="hash-link" aria-label="Direct link to Is SciChart.js free?" translate="no" title="Direct link to Is SciChart.js free?">â€‹</a>

For enterprise and commercial use, SciChart.js is paid software, however, version 3.2 - 4.x and upwards of SciChart.js has a **FREE Community License**, which is licensed for personal use, non-commercial use, educational use and limited-time commercial evaluations.

![](out_scichartv4/user-manual/licensing-scichart-js/getting-started-for-free/index_media/f0173787a207cc0a720ca619ab3e16cfe886b3c7.svg)tip

Just download and go with `npm install scichart`, and start developing with SciChart.js. No need to sign-up to the website, or speak to a salesperson.

### When can I use the free community edition of SciChart.js?<a href="https://www.scichart.com/documentation/js/v4/user-manual/licensing-scichart-js/getting-started-for-free/#when-can-i-use-the-free-community-edition-of-scichartjs" class="hash-link" aria-label="Direct link to When can I use the free community edition of SciChart.js?" translate="no" title="Direct link to When can I use the free community edition of SciChart.js?">â€‹</a>

The community license can be used for **personal use**, **educational use**, **non-commercial use**, and limited time **commercial evaluation**. The community license can be used locally, or deployed to live sites.

For more details on community licensing, including who can use it, features and restrictions, check out the <a href="https://www.scichart.com/community-licensing/" rel="noopener noreferrer" target="_blank">community licensing page</a>.

### What features & restrictions are in the free community license of SciChart.js?<a href="https://www.scichart.com/documentation/js/v4/user-manual/licensing-scichart-js/getting-started-for-free/#what-features--restrictions-are-in-the-free-community-license-of-scichartjs" class="hash-link" aria-label="Direct link to What features &amp; restrictions are in the free community license of SciChart.js?" translate="no" title="Direct link to What features &amp; restrictions are in the free community license of SciChart.js?">â€‹</a>

The SciChart.js community license is fully featured, has all the features of SciChart.js, all the performance of SciChart.js, all 2D/3D chart types, but displays a watermark.

The <a href="https://support.scichart.com/support/solutions/articles/101000482819-telemetry-watermark-in-community-licenses" rel="noopener noreferrer" target="_blank">watermark can be moved</a> (placed) in any corner of the chart, but can only be removed by purchasing a commercial license.

![](out_scichartv4/user-manual/licensing-scichart-js/getting-started-for-free/index_media/0f0874621de662630fd5d7f09c89455c4ace26e6.svg)warning

There are some time restrictions on the community license for deployed apps which are covered in the <a href="https://support.scichart.com/support/solutions/articles/101000482817-time-feature-restrictions-of-community-licensing" rel="noopener noreferrer" target="_blank">time and feature restrictions of the community license</a> page.

### What if I'm commercially evaluating SciChart.js, do I need to start a trial or sign-up?<a href="https://www.scichart.com/documentation/js/v4/user-manual/licensing-scichart-js/getting-started-for-free/#what-if-im-commercially-evaluating-scichartjs-do-i-need-to-start-a-trial-or-sign-up" class="hash-link" aria-label="Direct link to What if I&#39;m commercially evaluating SciChart.js, do I need to start a trial or sign-up?" translate="no" title="Direct link to What if I&#39;m commercially evaluating SciChart.js, do I need to start a trial or sign-up?">â€‹</a>

The community license may be used for commercial evaluation purposes for *a reasonable time period*. Just `npm install scichart` and try out our software.

![](out_scichartv4/user-manual/licensing-scichart-js/getting-started-for-free/index_media/af50301a53fa555616a9b2279e7e25a1d566cb1a.svg)info

For commercial evaluations, if you need help understanding SciChart.js capabilities, features, or if it can meet your requirements, contact <a href="https://www.scichart.com/contact-us/#tech-support" rel="noopener noreferrer" target="_blank">tech support</a>, use our <a href="https://chat.scichart.com" rel="noopener noreferrer" target="_blank">free AI Assistant</a> or ask a question on the <a href="https://www.scichart.com/questions" rel="noopener noreferrer" target="_blank">SciChart.js Forums</a>.

## When is a Commercial (Paid) License needed for SciChart.js?<a href="https://www.scichart.com/documentation/js/v4/user-manual/licensing-scichart-js/getting-started-for-free/#when-is-a-commercial-paid-license-needed-for-scichartjs" class="hash-link" aria-label="Direct link to When is a Commercial (Paid) License needed for SciChart.js?" translate="no" title="Direct link to When is a Commercial (Paid) License needed for SciChart.js?">â€‹</a>

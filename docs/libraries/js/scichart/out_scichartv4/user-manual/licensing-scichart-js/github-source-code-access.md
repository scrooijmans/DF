# Github Source Code Access

![](out_scichartv4/user-manual/licensing-scichart-js/github-source-code-access/index_media/af50301a53fa555616a9b2279e7e25a1d566cb1a.svg)info

To access the source-code for scichart.js, you will need a source-code enabled product. Products purchased from our store that enable SciChart.js source-code access include:

- SciChart Bundle 2D/3D Source (product code `SC-BUNDLE-SDK-ENTERPRISE`)

For purchasing, please visit <a href="https://www.scichart.com/shop" rel="noopener noreferrer" target="_blank">scichart.com/shop</a>. For pre-sales enquiries <a href="https://www.scichart.com/contact-us/#pre-sales" rel="noopener noreferrer" target="_blank">contact technical sales</a> and for licensing support post-sales please contact <a href="https://www.scichart.com/contact-us/#tech-support" rel="noopener noreferrer" target="_blank">technical support</a>.

If you are a SciChart.js Source-Code customer, you can now access the source code via our <a href="https://www.scichart.com/github-repository-for-scichart-source-code-customers/" rel="noopener noreferrer" target="_blank">Github repository</a>. To get access, please follow the steps below:

## Step 1: Activate your developer license<a href="https://www.scichart.com/documentation/js/v4/user-manual/licensing-scichart-js/github-source-code-access/#step-1-activate-your-developer-license" class="hash-link" aria-label="Direct link to Step 1: Activate your developer license" translate="no" title="Direct link to Step 1: Activate your developer license">â€‹</a>

You will need to activate a developer license for product code `SC-BUNDLE-SDK-ENTERPRISE` following the steps from [activating a paid developer license](https://www.scichart.com/documentation/js/v4/user-manual/licensing-scichart-js/activating-paid-licenses/)

## Step 2: Link your github account<a href="https://www.scichart.com/documentation/js/v4/user-manual/licensing-scichart-js/github-source-code-access/#step-2-link-your-github-account" class="hash-link" aria-label="Direct link to Step 2: Link your github account" translate="no" title="Direct link to Step 2: Link your github account">â€‹</a>

You can now link your github account in the <a href="https://www.scichart.com/my-account" rel="noopener noreferrer" target="_blank">scichart.com/my-account</a> page.

1.  Navigate to **My Account** - **Orders & Keys** - **Manage Licenses** for the source-code product you have purchased
2.  Under the **"License Details"** section, click the link **"Update your Github Account"**
3.  Submit a Github username, accept the terms and click submit

<img src="out_scichartv4/user-manual/licensing-scichart-js/github-source-code-access/index_media/e284b279a7cad57f7126d0f53e2477fdc93455a8.png" class="img_ev3q" decoding="async" loading="lazy" width="1431" height="974" alt="Link Github account to scichart source code order" />

<img src="out_scichartv4/user-manual/licensing-scichart-js/github-source-code-access/index_media/dc7c1608950fd11b571b2d9ff67abd447ff2c872.png" class="img_ev3q" decoding="async" loading="lazy" width="681" height="646" alt="Link Github account to scichart source code order" />

4.  You will receive an invite email **to the email associated with your Github account**. Click to accept this to be granted access to the repository.

5.  You should now be able to access the SciChart.js private Github repository at <a href="https://github.com/ABTSoftware/SciChartJS" rel="noopener noreferrer" target="_blank">github.com/ABTSoftware/SciChartJS</a>

The option to link your github account will only be available if you have a support-active, **activated source-code license** type (SciChart Bundle 2D/3D Source)

![](out_scichartv4/user-manual/licensing-scichart-js/github-source-code-access/index_media/0f0874621de662630fd5d7f09c89455c4ace26e6.svg)warning

If your support subscription expires and is not renewed, access to the github repos with SciChart source-code will be removed

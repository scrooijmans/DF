### Optional stackedGroupId

stackedGroupId: string

Sets the Stacked Group ID

description  
The Stacked Group Id is used to group [StackedColumnRenderableSeries](https://www.scichart.com/documentation/js/v4/typedoc/classes/stackedcolumnrenderableseries.html) inside a [StackedColumnCollection](https://www.scichart.com/documentation/js/v4/typedoc/enums/eseriestype.html#stackedcolumncollection) into vertical or horizontal stacked groups.

For a normal stacked column chart (stacks vertically), set [StackedColumnRenderableSeries.stackedGroupId](https://www.scichart.com/documentation/js/v4/typedoc/classes/stackedcolumnrenderableseries.html#stackedgroupid) inside a [StackedColumnCollection](https://www.scichart.com/documentation/js/v4/typedoc/enums/eseriestype.html#stackedcolumncollection) to the SAME value

For a column chart (stacked side by side), set [StackedColumnRenderableSeries.stackedGroupId](https://www.scichart.com/documentation/js/v4/typedoc/classes/stackedcolumnrenderableseries.html#stackedgroupid) inside a [StackedColumnCollection](https://www.scichart.com/documentation/js/v4/typedoc/enums/eseriestype.html#stackedcolumncollection) to DIFFERENT values

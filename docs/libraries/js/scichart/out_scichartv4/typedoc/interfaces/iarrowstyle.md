## Properties

### Optional fill

fill: string

The fill color of the arrow head.

note  
If `arrowStyle.headDepth = 0`, this property will be ignored

Default `transparent`

### Optional headDepth

headDepth: number

The depth of the arrow head **relative** to the `arrowStyle.headLength` property.

note  
If `1`, the arrowhead will be an isosceles triangle

note  
If `0.5`, the arrowhead will have sharp wings

note  
If `0`, the arrow will not have a base nor a fill

Default `0`

### Optional headLength

headLength: number

The length (height) of the arrow head in pixels Default `10`

### Optional headWidth

headWidth: number

The width of the arrow head in pixels Default `8`

### Optional stroke

stroke: string

The stroke color of the arrow head Default `#FFFFFF`

### Optional strokeThickness

strokeThickness: number

The stroke thickness of the arrow head in pixels Default `1`

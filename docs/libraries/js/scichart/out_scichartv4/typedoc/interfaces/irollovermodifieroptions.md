### Optional allowTooltipOverlapping

allowTooltipOverlapping: boolean

Sets if tooltips for multiple series are allowed to overlap. Default false

### Optional excludedSeriesIds

excludedSeriesIds: string\[\]

### Optional hitTestRadius

hitTestRadius: number

If this is set greater than the default of zero, the toolip will only show values for points in this radius, rather than all points on the vertical line

id: string

A unique Id for the [ChartModifierBase2D](https://www.scichart.com/documentation/js/v4/typedoc/classes/chartmodifierbase2d.html)

includedSeriesIds: string\[\]

modifierGroup: string

placementDivId: string

rolloverLineStroke: string

Sets the color of the vertical rollover line as an html color code

rolloverLineStrokeDashArray: number\[\]

rolloverLineStrokeThickness: number

Sets the thickness of the vertical rollover line

### Optional secondaryExecuteCondition

secondaryExecuteCondition: <a href="https://www.scichart.com/documentation/js/v4/typedoc/index.html#tmodifierexecutecondition" class="tsd-signature-type">TModifierExecuteCondition</a>

The secondary action execute condition that modifier should respond to

### Optional showAxisLabel

showAxisLabel: boolean

Sets if the axis label for the rollover Line should be shown. default false. Customize this futher after the modifier has been created by setting properties on rolloverModifer.rolloverLineAnnotation

### Optional showRolloverLine

showRolloverLine: boolean

Gets or Sets whether to show the vertical rollover line. Default true

### Optional showTooltip

showTooltip: boolean

Sets whether to show the tooltip. Default true

### Optional snapToDataPoint

snapToDataPoint: boolean

If True the [RolloverModifier](https://www.scichart.com/documentation/js/v4/typedoc/classes/rollovermodifier.html) line snaps to the nearest data-point of the first visible renderable series

### Optional tooltipDataTemplate

tooltipDataTemplate: <a href="https://www.scichart.com/documentation/js/v4/typedoc/index.html#trollovertooltipdatatemplate" class="tsd-signature-type">TRolloverTooltipDataTemplate</a> \| string

Sets the tooltipDataTemplate, which allows to customize content for the tooltip

### Optional tooltipLegendOffsetX

tooltipLegendOffsetX: number

Sets the legend X offset

### Optional tooltipLegendOffsetY

tooltipLegendOffsetY: number

Sets the legend Y offset

### Optional tooltipLegendTemplate

tooltipLegendTemplate: <a href="https://www.scichart.com/documentation/js/v4/typedoc/index.html#trolloverlegendsvgtemplate" class="tsd-signature-type">TRolloverLegendSvgTemplate</a> \| string

Sets the template for the legend

### Optional xAxisId

xAxisId: string

The XAxis Id to be used by annotations internal to the modifier. Set if you have multiple x axes and need to distinguish between horizontal/vertical, or stacked axes

### Optional xyDirection

xyDirection: <a href="https://www.scichart.com/documentation/js/v4/typedoc/enums/exydirection.html" class="tsd-signature-type">EXyDirection</a>

Defines the [Xy Direction](https://www.scichart.com/documentation/js/v4/typedoc/enums/exydirection.html) - whether the modifier works in X, Y or XY or neither direction, for vertical charts the behaviour could be inverted, for example for vertical chart with [RubberBandXyZoomModifier](https://www.scichart.com/documentation/js/v4/typedoc/classes/rubberbandxyzoommodifier.html) and [EXyDirection.XDirection](https://www.scichart.com/documentation/js/v4/typedoc/enums/exydirection.html#xdirection) the modifier works on Y axis

### Optional yAxisId

yAxisId: string

The YAxis Id to be used by annotations internal to the modifier. Set if you have multiple y axes and need the modifier to use something other than the first one.

# Interface IPointMarker3D

Defines the interface to a 3D PointMarker in SciChart - High Performance <a href="https://www.scichart.com/javascript-chart-features" class="external">JavaScript Charts</a>.

description  
PointMarkers may be displayed on [3D Renderable Series](https://www.scichart.com/documentation/js/v4/typedoc/interfaces/irenderableseries3d.html) to add scatter-points to charts.

remarks  
See derived types of [IPointMarker3D](https://www.scichart.com/documentation/js/v4/typedoc/interfaces/ipointmarker3d.html) for specific point-marker types.

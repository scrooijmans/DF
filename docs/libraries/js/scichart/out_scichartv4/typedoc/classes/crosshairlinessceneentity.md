# Class CrosshairLinesSceneEntity

Go to [JavaScript Charting Documentation](https://www.scichart.com/documentation/js/v4/intro)

Go to <a href="https://scichart.com/demo" target="_blank">JavaScript Chart Examples</a>

Go to <a href="https://github.com/abtsoftware/scichart.js.examples" target="_blank">SciChart.js on Github</a>

Draws a crosshair from the [location](https://www.scichart.com/documentation/js/v4/typedoc/classes/crosshairlinessceneentity.html#location) to the walls of the bounding box (defined by [worldDimensions](https://www.scichart.com/documentation/js/v4/typedoc/classes/crosshairlinessceneentity.html#worlddimensions)). Properties [stroke](https://www.scichart.com/documentation/js/v4/typedoc/classes/crosshairlinessceneentity.html#stroke), [strokeThickness](https://www.scichart.com/documentation/js/v4/typedoc/classes/crosshairlinessceneentity.html#strokethickness) and [antiAliased](https://www.scichart.com/documentation/js/v4/typedoc/classes/crosshairlinessceneentity.html#antialiased) can be used to customize the appearance of the line. The [CrosshairLinesSceneEntity](https://www.scichart.com/documentation/js/v4/typedoc/classes/crosshairlinessceneentity.html) inherits [BaseSceneEntity3D](https://www.scichart.com/documentation/js/v4/typedoc/classes/basesceneentity3d.html) and can be added to the [SciChart3DSurface](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichart3dsurface.html) via the [SciChart3DSurface.rootEntity](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichart3dsurface.html#rootentity) property.

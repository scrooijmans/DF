summary  
Defines the base class for Renderable Series Animations in SciChart's High Performance Real-time <a href="https://www.scichart.com/javascript-chart-features" class="external">JavaScript Charts</a>

remarks  
An Animation defines the animation that should take place on the start up e.g. as a Wave, Sweep, Fade animation etc...

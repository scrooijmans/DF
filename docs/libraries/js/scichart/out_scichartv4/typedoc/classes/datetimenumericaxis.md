A 2D Chart DateTime Axis type

description  
A DateTime axis is a specialized numeric axis that displays date and time values.

remarks  
Set a [DateTimeNumericAxis](https://www.scichart.com/documentation/js/v4/typedoc/enums/eaxistype.html#datetimenumericaxis) on the [SciChartSurface.xAxes](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartsurface.html#xaxes) or [SciChartSurface.yAxes](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartsurface.html#yaxes) property.

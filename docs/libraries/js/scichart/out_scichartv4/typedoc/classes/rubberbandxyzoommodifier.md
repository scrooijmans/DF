# Class RubberBandXyZoomModifier

Go to [JavaScript Charting Documentation](https://www.scichart.com/documentation/js/v4/intro)

Go to <a href="https://scichart.com/demo" target="_blank">JavaScript Chart Examples</a>

Go to <a href="https://github.com/abtsoftware/scichart.js.examples" target="_blank">SciChart.js on Github</a>

The RubberBandXyZoomModifier provides drag-rectangle to zoom behavior on a 2D [SciChartSurface](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartsurface.html) within SciChart - High Performance <a href="https://www.scichart.com/javascript-chart-features" class="external">JavaScript Charts</a>

remarks  
To apply the RubberBandXyZoomModifier to a [SciChartSurface](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartsurface.html) and add drag to zoom behavior, use the following code:

``` ts
const sciChartSurface: SciChartSurface;
sciChartSurface.chartModifiers.add(new RubberBandXyZoomModifier());
```

Animation of the zoom may be controlled via the [RubberBandXyZoomModifier.isAnimated](https://www.scichart.com/documentation/js/v4/typedoc/classes/rubberbandxyzoommodifier.html#isanimated), [RubberBandXyZoomModifier.animationDuration](https://www.scichart.com/documentation/js/v4/typedoc/classes/rubberbandxyzoommodifier.html#animationduration) and [RubberBandXyZoomModifier.easingFunction](https://www.scichart.com/documentation/js/v4/typedoc/classes/rubberbandxyzoommodifier.html#easingfunction) properties.

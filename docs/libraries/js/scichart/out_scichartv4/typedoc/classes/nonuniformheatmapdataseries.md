# Class NonUniformHeatmapDataSeries

Go to [JavaScript Charting Documentation](https://www.scichart.com/documentation/js/v4/intro)

A [NonUniformHeatmapDataSeries](https://www.scichart.com/documentation/js/v4/typedoc/classes/nonuniformheatmapdataseries.html) is a type of [BaseHeatmapDataSeries](https://www.scichart.com/documentation/js/v4/typedoc/classes/baseheatmapdataseries.html) which allows you to define non-uniform cell sizes and offsets for each cell row and column in the heatmap.

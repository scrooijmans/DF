# Class PieLabelProvider

Go to [JavaScript Charting Documentation](https://www.scichart.com/documentation/js/v4/intro)

summary  
A Label Provider for use with Pie Charts to allow customizing of segment labels To completely customise the label output, override [getSegmentText](https://www.scichart.com/documentation/js/v4/typedoc/classes/pielabelprovider.html#getsegmenttext) To just adjust the numeric formatting of the label, use numericformat, precision, prefix, postfix, or override [formatLabel](https://www.scichart.com/documentation/js/v4/typedoc/classes/pielabelprovider.html#formatlabel)

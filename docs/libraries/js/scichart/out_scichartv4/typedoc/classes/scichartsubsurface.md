# Class SciChartSubSurface

Go to [JavaScript Charting Documentation](https://www.scichart.com/documentation/js/v4/intro)

summary  
The [SciChartSubSurface](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartsubsurface.html) is the surface created within another surface

description  
It can be added using [SciChartSubSurface.createSubSurface](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartsubsurface.html#createsubsurface) method.

To update the positioning of the [SciChartSubSurface](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartsubsurface.html), use [SciChartSubSurface.subPosition](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartsubsurface.html#subposition); also you can call [SciChartSubSurface.updateSubLayout](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartsubsurface.html#updatesublayout) to refresh the layout of the sub-surface.

remarks  
It is not possible to have more than one level of nested sub-surfaces.

description  
A [SciChartSurface](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartsurface.html) can have zero to many [Annotations](https://www.scichart.com/documentation/js/v4/typedoc/interfaces/iannotation.html).

The Annotations are drawn using our WebGL / WebAssembly rendering engine, but some use SVG for maximum configurability. See derived types of [IAnnotation](https://www.scichart.com/documentation/js/v4/typedoc/interfaces/iannotation.html) such as [BoxAnnotation](https://www.scichart.com/documentation/js/v4/typedoc/classes/boxannotation.html), [LineAnnotation](https://www.scichart.com/documentation/js/v4/typedoc/classes/lineannotation.html) etc...

Use this collection to add and remove Annotations to the chart.

remarks  
Adding an Annotation to the chart causes it to automatically redraw. Note that annotations do not pariticpate in autoranging, meaning a chart will zoom to fit data and chart series but not annotations

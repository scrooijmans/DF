# Class SciChartOverview

Go to [JavaScript Charting Documentation](https://www.scichart.com/documentation/js/v4/intro)

Go to <a href="https://scichart.com/demo" target="_blank">JavaScript Chart Examples</a>

Go to <a href="https://github.com/abtsoftware/scichart.js.examples" target="_blank">SciChart.js on Github</a>

summary  
The [SciChartOverview](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartoverview.html) is the component that can control the visible range of the parent [SciChartSurface](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartsurface.html) using a slider <a href="https://www.scichart.com/javascript-chart-features" class="external">JavaScript Chart Library</a>

description  
To instantiate an overview use [SciChartOverview.create](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartoverview.html#create) method

remarks  
It is possible to specify options to customize style and behavior of the component.

# Class SciChartPieSurface

Go to [JavaScript Charting Documentation](https://www.scichart.com/documentation/js/v4/intro)

Go to <a href="https://scichart.com/demo" target="_blank">JavaScript Chart Examples</a>

Go to <a href="https://github.com/abtsoftware/scichart.js.examples" target="_blank">SciChart.js on Github</a>

summary  
The [SciChartPieSurface](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartpiesurface.html) is the root Pie and Donut Chart control in SciChart's High Performance Real-time <a href="https://www.scichart.com/javascript-chart-features" class="external">JavaScript Chart Library</a>

description  
To create a Pie chart using SciChart, declare a [SciChartPieSurface](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartpiesurface.html) using [SciChartPieSurface.create](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartpiesurface.html#create),

Next, add a pie segments by adding a [PieSegment](https://www.scichart.com/documentation/js/v4/typedoc/classes/piesegment.html) to the [SciChartPieSurface.pieSegments](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartpiesurface.html#piesegments) collection.

You can create a donut chart by setting the [SciChartPieSurface.holeRadius](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartpiesurface.html#holeradius) property.

You can add and configure a legend by setting the [SciChartPieSurface.legend](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartpiesurface.html#legend) property.

remarks  
It is possible to have more than one [SciChartPieSurface](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartpiesurface.html) on screen at the same time. [SciChartPieSurfaces](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartpiesurface.html) scale to fit the parent DIV where they are hosted. Use CSS to position the DIV.

# Class PolarZoomExtentsModifier

Go to [JavaScript Charting Documentation](https://www.scichart.com/documentation/js/v4/intro)

Go to <a href="https://scichart.com/demo" target="_blank">JavaScript Chart Examples</a>

Go to <a href="https://github.com/abtsoftware/scichart.js.examples" target="_blank">SciChart.js on Github</a>

The PolarZoomExtentsModifier provides double-tap or double-click to zoom-to-fit (Zoom Extents) behavior on a 2D [SciChartPolarSurface](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartpolarsurface.html) within SciChart - High Performance <a href="https://www.scichart.com/javascript-chart-features" class="external">JavaScript Charts</a>

remarks  
To apply the PolarZoomExtentsModifier to a [SciChartPolarSurface](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartpolarsurface.html) and add drag to zoom behavior, use the following code:

``` ts
const sciChartSurface: SciChartPolarSurface;
sciChartSurface.chartModifiers.add(new PolarZoomExtentsModifier());
```

Animation of the zoom extents be controlled via the [PolarZoomExtentsModifier.isAnimated](https://www.scichart.com/documentation/js/v4/typedoc/classes/polarzoomextentsmodifier.html#isanimated), [PolarZoomExtentsModifier.animationDuration](https://www.scichart.com/documentation/js/v4/typedoc/classes/polarzoomextentsmodifier.html#animationduration) and [PolarZoomExtentsModifier.easingFunction](https://www.scichart.com/documentation/js/v4/typedoc/classes/polarzoomextentsmodifier.html#easingfunction) properties.

# Class XAxisDragModifier

Go to [JavaScript Charting Documentation](https://www.scichart.com/documentation/js/v4/intro)

Go to <a href="https://scichart.com/demo" target="_blank">JavaScript Chart Examples</a>

Go to <a href="https://github.com/abtsoftware/scichart.js.examples" target="_blank">SciChart.js on Github</a>

The XAxisDragModifier provides scaling/panning behavior for X axis [AxisBase2D](https://www.scichart.com/documentation/js/v4/typedoc/classes/axisbase2d.html) within SciChart - High Performance <a href="https://www.scichart.com/javascript-chart-features" class="external">JavaScript Charts</a>

remarks  
To apply the XAxisDragModifier to a [SciChartSurface](https://www.scichart.com/documentation/js/v4/typedoc/classes/scichartsurface.html) and add scaling behavior, use the following code:

``` ts
sciChartSurface.chartModifiers.add(new XAxisDragModifier());
```

# Text and Multi-Line Labels

Text Labels and Multi-Line can be achieved in SciChart.js using the TextLabelProvider type.

For more info, see the [Text / String Axis](https://www.scichart.com/documentation/js/v4/2d-charts/axis-api/axis-types/text-string-axis) article under Axis Types.
